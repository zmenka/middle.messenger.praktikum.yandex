## Установка

- `npm install` — установка необходимых пакетов,
- `npm run dev` — запуск версии для разработчика,
- `npm run build` — сборка  продовой версии,
- `npm run start` — сборка и запуск продовой версии.
- `npm run lint` — запуск линтеров.
